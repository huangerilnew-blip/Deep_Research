- Exa mcp
  - 10元免费额度：2000次左右搜索
  - 分为简单工具，和高阶工具
- Tavily mcp
  - 优点：传统的 Google 搜索返回的是给“人”看的网页列表，而 Tavily 是专门为 **LLM（大语言模型）** 设计的,会对数据进行清清晰和过滤。
  - tavily_search（实时搜索）：让 AI 搜索互联网上的最新信息，不仅返回链接，还会返回针对 AI 优化的网页摘要。
  - tavily_extract（精准提取）：告诉 AI 一个 URL，它能直接抓取该网页的干净文本（剔除广告和杂质），方便 AI 进行总结或分析。
  - tavily_map` / `crawl（网站制图与爬取）： 深入探索某个网站的结构，获取更多层级的链接和内容。
  - 高级过滤： 支持设置搜索深度（Basic/Advanced）、过滤特定域名、按时间范围（如搜索近 3 天的新闻）等。
  - 每月有 1000 次免费额度
- Brave Search MCP
  - 每月有2000次的免费额度
  - brave_web_search` (通用网页搜索)：
    - **能力**：执行全局网页搜索，获取最新的新闻、文章和博客内容。
    - **特点**：支持分页和时效性控制（例如搜索过去 24 小时内的消息）。
  - brave_local_search` (本地服务搜索)：
    - **能力**：寻找附近的餐厅、商店、服务机构等。
    - **特点**：提供详细信息，如评分、地址、电话和营业时间。如果本地搜索无果，它会自动回退到通用网页搜索。
  - 没有对网页进行清晰，质量比较泛泛
- paper-search-mcp
  - 支持arXiv, PubMed 和 bioRxiv搜索，并且能够对内容做清洗解析，提供markdown格式的数据。
  - 在学术界，其整体反馈不错
  - search工具的返回结果
    - 论文标题 (Title)
    - 作者列表 (Authors)
    - 摘要 (Abstract)或 TL;DR（简短总结）
    - 发布日期与分类
    - PDF 链接或 DOI 链接
  - fetch 或 read类工具其返回结果
    - 尝试获取全文或详细元数据
- “搜索 + 网页即时解析”：Kindly
  - 会自动抓取该页面的Metadata 和全文摘要
  - 优势：它能把 arXiv 搜索和其他 Web 信息（如 GitHub 代码库、StackOverflow 讨论）整合在一起，提供最全的上下文。
